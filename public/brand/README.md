# Brand assets

Substitua os arquivos nesta pasta pelos assets oficiais da IAplicada:

- `logo.svg` — logo do site (usado em Header, Footer, OG image)
- `mariana.jpg` — foto da Mariana Marques (usada na seção Authority)
- `logos/` — (opcional) pasta com logos dos clientes reais, ex.:
  - `logos/natura.svg`
  - `logos/ambev.svg`
  - Depois edite `src/config/brand.ts` adicionando o campo `src` em cada cliente.
- `testimonials/` — (opcional) fotos dos depoentes.

Todos os caminhos são relativos à raiz (`/brand/logo.svg` no HTML).
